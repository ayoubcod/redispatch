# REdispatch.com | Documentation Hub

This repository contains the source code for the [REdispatch Engine](https://redispatchengine.com) website. 

REdispatch is a technical project focused on developing **deterministic engineering specifications** for the energy transition. This site serves as a documentation hub and educational resource for energy dispatch logic, BESS operations, and carbon accounting.

## Live Site
View the documentation at: [https://redispatchengine.com](https://redispatchengine.com)

## Tech Stack
- **HTML5/CSS3**: Custom-built responsive UI (no heavy frameworks).
- **JavaScript**: Vanilla JS for theme toggling and UI interactions.
- **SEO**: Fully optimized Metadata and Open Graph tags for technical sharing.

## Project Structure
- `/index.html`: Homepage & Project Overview.
- `/modules.html`: Catalog of engineering modules.
- `/methods.html`: Detailed deterministic logic and specifications.
- `/contact.html`: Connectivity and inquiry portal.
- `/assets/`: CSS styles, branding images, and icons.
